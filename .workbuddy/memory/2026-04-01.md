## 修复 break/continue 动画逐行高亮与变量同步显示

- 问题：CmdBreakPage 和 CmdContinuePage 的 activeLine 与实际执行语句不匹配，且变量值未与每步同步
- 修复方案：
  1. 将每次循环拆成多个"子步"（substep），每步对应一行代码高亮
  2. CmdBreakPage：5 个子步 → while(true)→if判断→placeAhead→moveForward→e-=3
  3. CmdContinuePage：4 个子步 → while条件→i++→if判断→吃包子(或 continue 跳回 while)
  4. 提取纯函数 computeNext / computeNextContinue，改用 tick state 驱动 autoPlay（避免 setInterval 闭包捕获旧 state 的 bug）
  5. runtimeBadges 每步精确显示当前变量值，runtimeHint 显示执行说明
- 验证：pnpm lint ✅ pnpm build ✅（471.98 kB）
- 提交：fix: 修复 break/continue 动画逐行高亮与变量值同步显示（commit: 5124b01）
